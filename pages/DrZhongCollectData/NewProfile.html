New Faculty Profile

  1. 本页面基于HTML5特性搭建，请使用最新版本的Chrome, Firefox, Edge或Safari进行访问
  2. 点击"Submit"后，浏览器会生成一个下载，下载文件自动命名为"教师姓名+当前学校.json"，如：James_Bond_Harvard University.json
  3. 本系统通过表单最后的电子签名识别填表人，请务必记得填写正确的姓名然后提交
  4. 请妥善保存下载的数据文件，待一个学校/系的全部教师填写完毕后，以学校为单位打包提交给指导教师。压缩包命名规范为“yymmdd_填表人姓名_学校名称”，如“190701_张三_harvarduniversity.zip”

I. BASIC INFORMATION 基本信息

NOTE:

  1. First name 亦称为 given name; last name 亦称为 surname
  2. 若无法判断出生国家（Birth country），请使用该教师本科毕业院校所在国家填写。
  3. sex 指的是该教师的生物学性别。若无法判断，请选择Unknown。
  4. 若找不到该教师的出生年份，请使用其本科毕业年份减去22作为出生年份估计。

II. RESEARCH FIELDS 研究领域

NOTE:

  1. 研究领域通常可在该教师的院系主页或索引页找到。若没有，请在Google Scholar查找该教师的个人简历。
  2. 若该教师披露的研究领域与列表选项不完全匹配，请选择最相近的选项，或联系指导教师。
  3. 推荐按照披露的顺序依次填写。若领域多于3个，则对于研究领域1（必填项）请填写最能代表该教师研究的领域。

III. EDUCATION 教育背景

NOTE:

  1. 对于不熟悉的学校，请配合实用链接中的世界大学名录查找并填写。
  2. 对于非拉丁字母命名的学校（如中国、日本等的学校），请使用CV中记录的学校英文名称填写。
  3. 若该教师拥有多个同水平学位（如双博士），请优先填写经济、金融类有关学位。
  4. Major字段，分割线上方为优先选项，若该教师的专业不属于优先选项，则请从下方的分类中选择一个
  5. Major字段，分割线下方的选项基于美国国会图书馆学科分类（点击查看）
  6. 对于不确定的学科，请查询美国国会图书馆学科分类然后填写
  7. 当然，我们预期非优先选项是少数 :)

IV. FULL-TIME: POST-DOCTORAL RESEARCHER 全职博士后

NOTE:

  1. 请按照时间逆序填写。
  2. Since 指该段经历开始的年份，Until 指经历结束的年份。
  3. 若该教师目前仍未结束某段经历，请在 Still 字段选择 Yes，且 Until 字段填写 2019
